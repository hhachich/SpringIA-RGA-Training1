Bienvenue sur notre modèle RAG

C'est un projet développé avec Spring Boot conçu pour tester la technologie de Retrieval-Augmented Generation (RAG). Notre objectif est d'enrichir un modèle de langage, tel que Mistral, avec des données issues de documents PDF afin d'améliorer ses capacités de génération et de compréhension.

Le projet se compose de deux principales fonctionnalités :

  • Chat : Cette fonctionnalité vous permet de discuter en temps réel avec notre modèle de langage enrichi. Grâce aux informations contenues dans les PDF chargés.
  • Upload : Cet onglet vous permet de charger des fichiers PDF dans notre système. Les données extraites de ces documents sont utilisées pour enrichir le modèle de langage, améliorant ainsi ses performances et ses réponses.
